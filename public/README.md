# public/ — image assets

Drop the two image files here (exact names matter):

| File         | What it is                                    |
| ------------ | --------------------------------------------- |
| `logo.png`   | The Muzium Negara logo (shown in the header)  |
| `kancil.png` | The Kancil mouse-deer character (avatar + welcome portrait) |

Until they're added, the app falls back to a built-in wordmark and a simple
face, so it still builds and deploys. Add the files and redeploy, no code change
needed.

You can also delete this README once the images are in.
